# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML5, CSS3, and JavaScript.

## Features

- Responsive design that works on all devices
- Dark professional theme
- Interactive project carousel
- Smooth scrolling navigation
- Contact form
- Skills visualization
- Experience timeline
- Project showcase

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/personal-site.git
```

2. Open `index.html` in your browser to view the website.

## Project Structure

```
personal-site/
├── index.html          # Main HTML file
├── styles.css          # CSS styles
├── script.js           # JavaScript functionality
├── README.md           # Project documentation
└── .gitignore         # Git ignore file
```

## Customization

- Update personal information in `index.html`
- Modify styles in `styles.css`
- Add or remove projects in the projects section
- Update experience and skills as needed
